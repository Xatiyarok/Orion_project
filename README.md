# Project Orion

## Overview

Project Orion is a custom GNSS receiver hardware platform designed around the u-blox MAX-M10S GNSS receiver and the ESP32-S3 microcontroller.

The objective of this project is not only to design a functional GPS receiver PCB, but also to document the complete engineering process behind a modern RF embedded system.

This repository contains the complete hardware development process, including:

- System architecture
- RF design considerations
- PCB layout
- Design decisions
- Improvements
- Firmware interface

## Objectives

- Design a custom GNSS receiver PCB.
- Learn RF PCB layout techniques.
- Understand GNSS receiver architecture.
- Integrate the receiver with an ESP32-S3.
- Document the engineering process from concept to PCB.

## Hardware

Main components:

- u-blox MAX-M10S
- ESP32-S3-WROOM-1U
- U.FL RF connector
- External GNSS antenna
- 3.3 V regulated power supply
